Simple steps to create your own mod for Diablo 2 (1.10+):

  • Select the features you want below
  • Click on blue button below to download zip file with options chosen
  • Extract zip file into Diablo 2 program folder. (This should add a data folder within the program folder)
  • Start Diablo 2 game with the following options '-direct -txt' passed to shortcut or executable file
  • Enjoy!

* Always make a backup of your character before trying out these mods

Replaces an existing skill to summon an extra sidekick for your hero.
Like the iron golem, you will need to choose a metal item to summon your sidekick.
Your side kick will inherit the properties of the item.
Unlike the iron golem, your sidekick will be lost when you exit the game
(TIP: do not use expensive items when creating your sidekick)
In addition, the sidekick will have the following properties: (where clvl = your character level)

  • Life: Same as character
  • Defense: increases with character defense level
  • Attack Rating: increases with character attack rating
  • Item normal damage: Maximum damage of non missile weapon currently used
  • Crushing blow: 25% chance
  • Natural Resistance (max level 20) level = 1+(clvl/3)

A new cube recipe will be added so you can access this skill

  • 1 key + 1 boots -> small charm (+1 to chosen skill)
