Simple steps to create your own mod for Diablo 2 (1.10+) or Diablo 2 Ressurected:

  • Step 1: Choose your product
  • Step 2: Select the features you want below
  • Step 3: Click on blue button below to download zip file with options chosen
  • Step 4: Extract zip file into Diablo 2 program folder. (This should add a data folder within the program folder)
  • Step 5: Start Diablo 2 game with the following options '-direct -txt' passed to shortcut or executable file
  • Step 4: Extract zip file into Diablo 2 Resurrected program folder. (This should add a mod folder within the program folder)
  • Step 5: Start Diablo 2 Resurrected game with the following options '-mod diymod -txt' passed to shortcut or executable file
  • Enjoy!

* Always make a backup of your character before trying out these mods

Adds extra Horadric Cube recipes. The recipes are:

  • 1 key + 1 socketed item -> unsocket (keep gems)
  • 1 boots + 1 stamina potion -> same boots +50% walk/run speed
  • 1 key + 1 thawing potion -> Rugged small charm
  • 1 charm + 1 town portal scroll -> same charm (+1 teleport)
  • 1 charm + 1 mana potion -> same charm (+1 all skills)
  • 1 charm + 1 thawing potion -> same charm (+1 corpse explosion)

Adds extra Horadric Cube recipes.

  • Convert items to rare type with random statistics
  • Duplicate items

The recipes are:

  • 1 weapon + 1 identify Scroll + 1 thawing potion -> rare weapon of same type
  • 1 armour + 1 identify Scroll + 1 thawing potion -> rare armour of same type (also works on rings and amulets)
  • 1 weapon + 2 health potions -> duplicate item
  • 1 armour + 2 health potions -> duplicate item
  • 1 charm + 2 health potions -> duplicate item
